# 🐍 Estudos de Python - Python 2025 Teo Me Why

Este repositório foi criado para documentar a minha evolução no de curso de Python do canal [**Teo me Why**](https://www.youtube.com/watch?v=OeKzVjiiRm4&list=PLvlkVRRKOYFSpRkqnR0p2A-eaVlpLnN3D). O objetivo é aplicar boas práticas de desenvolvimento, utilizando um fluxo de trabalho baseado em *branches*, *commits* padronizados e aprovação de *Pull Requests*.

## 🎯 Objetivo
Aprender os fundamentos da linguagem Python e progredir para análise e manipulação de dados, mantendo um histórico de código limpo e organizado.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **Controlo de Versão:** Git & GitHub
* **Ambiente:** VS Code
